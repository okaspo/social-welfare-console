# Antigravity Council Review Protocol

機能実装、仕様変更、または `/review` コマンドが実行された際、本スキルを実行してください。

## レビュー手順

1. **評議会の招集 (Resources)**
   - `resources/personas.md` を読み込み、そこに定義された「10人の専門家」の視点をシミュレーションしてください。
   - 各メンバーの「Focus（焦点）」と「Check（確認事項）」に基づき、提案されたコードや仕様を厳しく監査してください。

2. **評価の出力 (Examples)**
   - 監査結果は、必ず `examples/output.md` に定義されたテーブルフォーマットに従って出力してください。
   - 言語は**日本語**を使用してください。
   - 判定（Verdict）は明確に下してください。
